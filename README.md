# Homebrew Tap

Custom Homebrew formulas for lance0 tools.

## Installation

```bash
brew tap lance0/tap
brew install ahab
```

Or directly:

```bash
brew install lance0/tap/ahab
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [ahab](https://github.com/lance0/ahab) | Interactive Docker cleanup TUI |
