# Homebrew Tap

Custom Homebrew formulas for lance0 tools.

## ttl

Modern traceroute/mtr-style TUI with hop stats and ASN/geo enrichment.

![ttl screenshot](https://raw.githubusercontent.com/lance0/ttl/master/ttlss.png)

## Installation

```bash
brew tap lance0/tap
brew install ttl
```

Or directly:

```bash
brew install lance0/tap/ttl
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [ttl](https://github.com/lance0/ttl) | Modern traceroute/mtr-style TUI with hop stats and ASN/geo enrichment |
| [ahab](https://github.com/lance0/ahab) | Interactive Docker cleanup TUI |
| [kaioken](https://github.com/lance0/kaioken) | High-performance HTTP load testing tool with real-time TUI |
